This repo creates a posgreSQL table based on Los Angeles County's CAMS dataset and updated addresses in the City of Los Angeles.

It is assumed you have the following layers as tables in your PostGIS database. Here are some downloads:

<ul>
  <li><a href="https://lacounty.maps.arcgis.com/home/item.html?id=cdd4c011519849caa62286044f1d31c9">CAMS_Points</a></li>
  <li><a href="https://egis-lacounty.hub.arcgis.com/documents/parcels/about">Parcels</a></li>
  <li><a href="https://egis-lacounty.hub.arcgis.com/datasets/la-county-city-boundaries">City Boundaries</a></li>
  <li><a href="https://data.lacity.org/City-Infrastructure-Service-Requests/Addresses-in-the-City-of-Los-Angeles/4ca8-mxuh">Addresses in the City of L.A.</a></li>
</ul>
